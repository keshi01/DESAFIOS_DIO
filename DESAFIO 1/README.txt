
# Desafio 1
## Descrição
Otimizar o Sistema Bancário previamente desenvolvido com o uso de funções Python.
Refatorar o código existente, dividindo-o em funções reutilizáveis, facilitando a manutenção e o entendimento do sistema como um todo.
## Objetivo
 O objetivo é implementar três operações essenciais:

    + Depósito;
    + Saque;
    + Extrato;
    + Saida
Onde o Saque não pode ser feito quando o saldo estiver negativo, ter um limite de R$500,00, e não ser realizado mais de 3 vezes.


## Conhecimento
 O sistema será desenvolvido para um banco que busca monetizar suas operações. Durante o desafio, tento aplicar os seguintes conhecimentos em programação Python:

    	-LISTAS;
    	-TUPLAS;
    	-CONJUNTOS;
	-DICIONÁRIOS;
	-FUNÇÕES.
